交ぜ書き変換
************

ファイル :file:`skk-mazegaki.el` をインストールすると、交ぜ書き変換が可能となります。

  - き車 → 汽車
  - き者 → 記者
  - き社 → 貴社

インストール方法などは、次の投稿を参考にしてください。
